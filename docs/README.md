## 🎯 Platform Overview

NuttyFans is a next-generation global creator monetization platform that empowers content creators with sophisticated tools to build sustainable income streams. The platform supports both SFW and adult creators with strict category segregation.

### Key Features

- **Multi-Revenue Streams**: Subscriptions, PPV, tips, paid DMs, live streaming
- **Premium Experience**: Hidden.com-inspired editorial aesthetic
- **Fair Economics**: 15% commission with weekly payouts
- **Global Reach**: PWA-first web application with worldwide accessibility
- **Trust & Safety**: Mandatory KYC, AI moderation, watermarking

### Tech Stack

- **Frontend**: Next.js 14+, Tailwind CSS, shadcn/ui
- **Backend**: Next.js API Routes, Prisma ORM
- **Database**: PostgreSQL (Neon)
- **Storage**: AWS S3 + CloudFront CDN
- **Payments**: Stripe Connect
- **Video**: Mux for processing and streaming
